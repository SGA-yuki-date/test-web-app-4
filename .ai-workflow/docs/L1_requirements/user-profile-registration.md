# ユーザー基本情報登録 要求

## 概要

ログイン後のエンドユーザーが、健康記録・健康状態の分析に必要なユーザー基本情報（生年月日・性別）を登録できること。

## 受け入れ条件

- AC-001: ログイン後、ユーザー基本情報が未登録の場合にユーザー基本情報登録画面が表示されること
- AC-002: ユーザー基本情報登録画面には「生年月日(yyyy/mm/dd)」と「性別」の入力フィールドと登録ボタンが表示されること
- AC-003: ユーザーは「生年月日(yyyy/mm/dd)」と「性別」を入力し、登録ボタンを押下することでユーザーの基本情報を登録できること
